STATE: 🔴 Issues (M1 gating now blocks by supertrend_conflict; needs next tuning)
ROLE:
- Core signal computation:
  - Pulls MT5 rates
  - Computes indicators
  - Determines HTF bias
  - Computes BOS/breakout, retest, proximity, RR
  - Produces “signal package” + context + gates + blocked_by

INPUTS:
- config.json
- MT5 market data (rates)

OUTPUTS:
- Signal package dict (consumed by mentor_executor.py)

TECHNICAL INDICATORS USED:
- ATR (volatility / distance normalization)
- Bollinger Band width (BB upper - BB lower) and bb_width_atr = width/ATR
- Supertrend (trend direction + line value)
- (Optional telemetry) ADX, DI+/- (used for diagnostics if present)

TOOLS/LOGIC USED:
- Gate evaluation:
  bias_ok, supertrend_ok, vol_expansion_ok, bos_break_ok, retest_ok, proximity_ok, rr_ok
- blocked_by aggregation

CURRENT NUMBERS (tuning-critical; why they matter):
- bb_width_atr_min_m1 = 0.65 (M1 vol-exp threshold override) -> intended to prevent no_vol_expansion from dominating.
- Supertrend conflict currently blocks too aggressively on M1 (micro pullbacks flip ST).
- BOS break requirement currently yields no_bos_break frequently on M1 (needs review after supertrend is fixed).

CURRENT OBSERVED OUTCOME:

  supertrend_conflict + no_bos_break (and rr_too_low often)  [evidence: 20 แท่งล่าสุด.txt]

4) TECHNICAL INDICATORS & TOOLS (CONSOLIDATED)
================================================================================
Indicators (core):
- ATR: volatility normalization, SL/TP basis, distance normalization.
- Bollinger Band width: volatility expansion proxy; bb_width_atr = width / ATR.
- Supertrend: directional filter; conflict check against HTF bias.
- BOS (Break of Structure): structure breakout confirmation.
- Retest + Proximity: entry quality filter (distance-to-level and score threshold).
- RR: risk/reward feasibility.

Tools:
- commissioning_runner.py: event-based evaluation
- logs/*.jsonl: observability storage
- tools/collect_events.py: sampling collector
- tools/blocker_stats.py: blocker analytics / gate breakdown
